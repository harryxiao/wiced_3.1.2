)

Initiate a SmartBridge connection with a remote Bluetooth Smart device.

Note
This function creates a connection with the specified remote device.
  • If Bond Information is loaded, the socket uses the Long-Term Key (LTK), Random Number (RAND), and Encryption Diversifier (EDIV) of the Bond Information to encrypt the connection. Call wiced_bt_smartbridge_set_bond_info() before connecting to load Bond information.
  • If Pairing is enforced, the socket initiates Pairing Request and a new Bond is created. Call wiced_bt_smartbridge_enable_pairing() before connecting to initiate Pairing Request.
  • If Attribute Cache is enabled, the socket starts a discovery of the server's Attribute information.
Warning
  • Callback functions run on the context of WICED_NETWORKING_WORKER_THREAD.
  • If Pairing and Attribute Cache are enabled, this function may block for a few seconds.
Parameters
[in,out]socket: pointer to the socket to create the connection
[in]remote_device: remote device to connect
[in]settings: connection settings
[in]disconnection_callback: callback function that is called when the connection is disconnected by remote device
[in]notification_callback: callback function that is called when a GATT notification or indication is received from the remote device
Returns
wiced_result_t
