wiced_result_t wiced_bt_packet_delete ( wiced_bt_packet_t *  packet)

Delete a WICED Bluetooth packet.

This function returns the packet's memory space back to the source, allowing for reuse.

Parameters
packet: The pointer to the packet to delete
Returns
WICED_SUCCESS : on success; WICED_BADARG : if bad argument(s) are inserted; WICED_ERROR : if an error occurred.
wiced_result_t wiced_bt_packet_get_data ( const wiced_bt_packet_t *  packet,
uint8_t **  data,
uint32_t *  current_data_size,
uint32_t *  available_space 
)

Get a pointer to the packet data.

This function retrieves a pointer to the start of the data section in the packet. It also returns the current data size and the remaining data space in the packet.

Parameters
packet: The pointer to the packet
data: A pointer that will receive the pointer to the start of the data section in the packet
current_data_size: A pointer that will receive the size of the data in the packet in bytes
available_space: A pointer that will receive the available data space in the packet in bytes
Returns
WICED_SUCCESS : on success; WICED_BADARG : if bad argument(s) are inserted; WICED_ERROR : if an error occurred.
