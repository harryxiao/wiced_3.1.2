wiced_result_t wiced_bt_did_enable ( const wiced_bt_did_mandatory_attributes_t *  mandatory_attributes,
const wiced_bt_did_optional_attributes_t *  optional_attributes 
)

Enable the Bluetooth Device ID (DID) Profile.

This function adds an SDP record for the Device ID Profile with the attributes provided.

Parameters
mandatory_attributes: A pointer to a structure containing mandatory attributes of the Device ID SDP record
optional_attributes: A pointer to a structure containing optional attributes of the Device ID SDP record
Warning
Only mandatory attributes are supported presently.
Returns
WICED_SUCCESS : on success; WICED_BADARG : if bad argument(s) are inserted; WICED_ERROR : if an error occurred.
